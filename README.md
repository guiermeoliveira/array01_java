
# Lista de Exercícios de Arrays (Java)

*Repositório dedicado à resolução de exercícios de Lógica de Programação (LOP) focados em manipulação de Arrays e ArrayLists. Este projeto faz parte da minha jornada de aprendizado em Análise e Desenvolvimento de sistemas.*

### 🚀 Tecnologias e Ferramentas
- Linguagem: Java (JDK 25+)

- Paradigma: Orientação a Objetos e Lógica Imperativa

- Interface: Terminal/Linha de Comando (CMD)

- Versionamento: Git & GitHub

 ---

### 🛠️ O que foi praticado:
**Os exercícios cobrem conceitos fundamentais de estrutura de dados em memória, incluindo:**

*1. Manipulação de Arrays Estáticos: Criação, iteração e acesso via índices.*

*2. ArrayLists Dinâmicos: Uso de métodos como .add(), .get(), .remove() e .size().*

*3. Lógica de Comparação: Algoritmos para encontrar valores máximos e mínimos (usando Float.MAX_VALUE e Float.MIN_VALUE).*

*4. Entrada e Saída de Dados: Uso de biblioteca de IO para interação via terminal.*

*5. Formatação de Saída: Criação de tabelas e bordas formatadas com String.format() e .repeat().*

---

### 💻 Como executar:
- **Clone o repositório:**
`CLI: git clone https://github.com/guiermeoliveira/LOP_Arrays.git`

- **Navegue até a pasta:**
`CLI: cd LOP_Arrays` 

- **Compile e execute os arquivos .java desejados:**
`CLI: java <NomeDoArquivo>.java`
